EpsilonWeb Scripting Demos

EpsilonWeb is a web runtime to execute Epsilon as a server-side scripting language.

Demos

  • cities.egl: displays the number and the name of the cities in the cities.graphml file.
  • import.egl: displays the number of meta-classes in the Ecore metamodel.
  • index.egl: displays a list of all the classes in the Ecore metamodel. Also, it displays information for each meta-class when you click on its name.
